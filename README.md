# Debug Controls

F1 - Activate/Deactivate Debug mode
3 - Set lives to 0
LEFT CLICK (while Debugging) - Activates mouse joints


#Keyboard Controls

DOWN ARROW - Compress/Decompress spring
LEFT ARROW - Moves left flipper
RIGHT ARROW - Moves right arrow
R - Reset ball
SPACE - Play/Play again
